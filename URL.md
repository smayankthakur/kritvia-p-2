## Auto-Generation

This file is automatically generated from the route structure in `/app`. To regenerate:

```bash
npm run generate-routes
```

The script scans `/app` for all `page.tsx` files and updates:
- `URL.md`
- `lib/routes.ts`
- `app/sitemap.ts`
- `lib/navigation.ts`
